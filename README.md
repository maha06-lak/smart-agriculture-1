🌱 Smart Agriculture System

An intelligent web-based agriculture platform that helps farmers make better decisions using Artificial Intelligence and Machine Learning. The system provides plant disease detection, irrigation recommendations, and soil-based crop suggestions.

🚀 Features
🌿 Plant Disease Detection using Vision Transformer (ViT)
💧 Irrigation Recommendation using GCN + SVM
🌱 Soil Analysis & Crop Suggestion using InceptionV3
📊 Real-time prediction through a user-friendly web interface
📱 Accessible via mobile and desktop
🧠 Technologies Used
Python
Machine Learning & Deep Learning
Flask (Backend)
HTML, CSS, JavaScript (Frontend)
Libraries: TensorFlow / PyTorch, scikit-learn, OpenCV, NumPy, Pandas
🏗️ System Architecture

The system consists of three main modules:

Disease Detection Module
Uses Vision Transformer to analyze leaf images and detect diseases.
Irrigation Module
Uses Graph Convolutional Network (GCN) and Support Vector Machine (SVM) to recommend irrigation levels.
Soil Analysis Module
Uses InceptionV3 CNN to classify soil types and suggest suitable crops.
